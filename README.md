## Task 1: Even or Odd Checker

This Python script prompts the user to enter a number.  
It checks whether the number is even or odd and prints the result accordingly.

**Example Output:**

**Output1:**

Enter a number: 7

7 is an odd number

**Output2:**

Enter a number: 12

12 is an even number
